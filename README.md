<a href="https://www.djangoproject.com/">Django 2.2</a>

<h3>Installation</h3>

Install crispy forms
```
pip3 install django-crispy-forms
```

Create Database
```
python3 manage.py makemigrations
```
```
python3 manage.py migrate
```
Run Server
```
python3 manage.py runserver
```
Visit
```
127.0.0.1:8000
```
