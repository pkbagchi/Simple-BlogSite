<h3>Requirements</h3>
<a href="https://www.python.org/">Python3  </a>
<a href="https://www.djangoproject.com/">Django 2.2</a>

<h3>Installation</h3>

Create Database
```
python3 manage.py migrate
```
Run Server
```
python3 manage.py runserver
```
Visit
```
127.0.0.1:8000
```
